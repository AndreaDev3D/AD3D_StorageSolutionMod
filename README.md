# AD3D_StorageSolutionMod

This is a mod for both Subnautica and Subnautica Below Zero that adds a set of storage solutions to the buildable menu.

## Installation

To install, you need **BepInEx** and **Nautilus**.

1. Download and unzip the mod folder.
2. Place the unzipped folder into your `BepInEx\plugins` directory.
